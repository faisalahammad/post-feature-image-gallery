Post Feature Image Gallery create a responsive gallery from your recent post thumbnail with prettyPhoto.

== Description ==

Post Feature Image Gallery is responsive mobile friendly image gallery. You could order image gallery by **ASC or DESC** from plugin's widget settings page.

Plugin Features:

* You could add as many images as you want.
* You could set gallery colmuns.
* You could order by ASC or DESC.
* Gallery will not include any post if it hasn't feature image.
* You could add your custom gallery title.
* You could generate shortcode and insert any post or page (Coming soon)
* You could exclude any specific post by post ID (Coming soon) 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/post-feature-image-gallery` directory, or install the plugin through the WordPress plugins screen directly by searching `post feature image gallery`.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Apperence->Widget and add `Post Feature Image Gallery` on your widget area.
4. Configure the widget and save it.


== Frequently Asked Questions ==

= Does it mobile responsive? =

Yes, it's fully mobile responsive and you could also set how many columns will show on the mobile screen. By default it'll show 2 columns from 414px screen size.

= Does it support Custom Post type? =

Not right now but on next release, it will support on just Premium version of this plugin.

= May I add post link on the image pop up? =

Yes, after next release, you could add.
